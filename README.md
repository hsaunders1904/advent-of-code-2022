# Advent of Code Solutions 2022

https://adventofcode.com/2022/

## Build & Run

Requires CMake >= 3.18 and C++-20.

```console
cmake -B build
cmake --build build
```

```console
./build/bin/day<XX> data/day<XX>.txt
```

## Tests

```console
./build/bin/aoc-test
```

## Benchmarks

```console
cd build
./bin/aoc-benchmarks
```
